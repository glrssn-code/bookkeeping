## 项目概述

- **技术栈**：Electron + Vite + React + Ant Design + ECharts
- **功能**：记账、月度统计、Excel/JSON导入导出
- **端口**：http://localhost:5173

## 布局

- 左侧 (380px)：日历卡片
- 中间 (380px)：类型选择 + 类别网格 + 金额输入
- 右侧 (320px)：当日记录列表
